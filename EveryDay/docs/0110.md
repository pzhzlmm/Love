##3.	运算符%的作用是什么? 有什么实际的应用?
作用：取余, 取模
应用：1）判断奇偶数, n % 2 == 0, 说明n是偶数, n % 2 != 0 说明n是奇数
 2）对一个数取余，结果总是小于这个数, n % m 结果总是[0~m) 让一个未知数落在已知范围内.
 3) n % m 如果结果为0, 说明n能被m整除. 如果结果不为0, 说明n不能被m整除.
